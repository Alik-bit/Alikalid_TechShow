Projects

Zynq LED Interrupt Controller

Built an embedded system using Zynq PS+PL architecture. Used push buttons to trigger external interrupts for controlling four LED counters. Configured AXI interconnects for seamless communication between the processing system and programmable logic.


Automated Water Irrigation System

Designed and soldered a custom PCB integrating XKC-Y25-PNP, moisture sensors, and relays. Used ESP32 to control water pumps based on soil moisture and tank level readings. Implemented wireless communication and error handling for robust automation.


UART Communication System

Created a UART interface to convert decimal to hexadecimal and handle edge cases. Developed 4-stage parser: numeral conversion, A–F handling, invalid entry filtering, prefix processing.


RGB LED Duty Cycle Control

Configured Output Compare pins to drive RGB LEDs with PWM. Mixed primary colors using variable duty cycles and timer settings. Analyzed visual output and fine-tuned timing for desired color blending.
